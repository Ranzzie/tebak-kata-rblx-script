# ranzziekbbi.lua
Script helper pencari kata untuk game Roblox dengan UI modern, auto-detect huruf dari UI/console, auto-type, speed control, dan reset/self-destruct.

## Fitur Utama
- Auto-detect huruf petunjuk
- Saran kata berdasarkan prefix
- Auto type kata (skip prefix yang sudah kamu ketik)
- Speed slider (`F` cepat, `S` lambat)
- Tombol `Troll`, `Reset`, dan `NUKE` (self-destruct)

## Loadstring
```lua
loadstring(game:HttpGet(""))()
```
